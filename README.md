# Chat-with-pdf-system
Chat with PDF is an AI-powered application that allows users to interact with PDF documents in a conversational way. Instead of manually searching through long documents, users can simply ask questions and get relevant answers instantly.
Features:
Upload and process any PDF document
 Ask questions in plain English
 Semantic search using vector embeddings
 Fast similarity search with FAISS
 Context-aware answers from document content
 Runs easily on Google Colab
